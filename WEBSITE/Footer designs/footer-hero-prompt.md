HERO FOOTER GENERATION MODE

Design a footer that behaves like a final cinematic hero section rather than a traditional utility footer.

The footer should feel emotionally memorable, visually dominant, and intentionally climactic — as if the experience reaches one final statement before ending.

This is not a simple link container.
It is the closing scene of the website.

The footer must feel:
- cinematic
- commanding
- immersive
- emotionally resonant
- premium
- atmospheric
- visually iconic
- compositionally powerful

Before generating:
- analyze the current project's visual identity
- study the typography system
- understand layout rhythm and spacing philosophy
- preserve the existing visual language and atmosphere
- detect the emotional tone of the website

The footer should:
- function as a final visual payoff
- create emotional closure
- leave a lasting impression
- feel intentionally staged like a hero section
- reinforce the project's identity at maximum strength

Focus heavily on:
- large-scale typography
- dramatic composition
- visual storytelling
- controlled contrast
- atmosphere
- spatial hierarchy
- cinematic spacing
- emotional framing
- premium restraint
- immersive structure

Possible hero-footer elements:
- oversized statement typography
- massive brand/title presence
- cinematic imagery or silhouette usage
- architectural grid layouts
- layered typography compositions
- emotionally weighted messaging
- dramatic negative space
- immersive transitions from content into footer

Avoid:
- generic corporate footer layouts
- cluttered navigation blocks
- startup CTA spam
- excessive link dumping
- random social icon walls
- overcomplicated interactions
- visual noise
- trendy UI gimmicks
- disconnected sections

Typography should feel:
- iconic
- cinematic
- deliberate
- emotionally weighted
- visually commanding

Interactions should:
- feel subtle but immersive
- reinforce cinematic presentation
- avoid distracting motion
- prioritize atmosphere over novelty

The footer must still remain integrated with the site's architecture and should not feel visually disconnected from the rest of the experience.

Do not redesign the website.
Do not abandon the project's established design philosophy.
Do not force aesthetics unrelated to the current interface system.

Output:
1. Hero-footer concept
2. Composition structure
3. Typography strategy
4. Emotional direction
5. Visual hierarchy
6. Spacing philosophy
7. Interaction behavior
8. Responsive behavior
9. Architectural integration notes
