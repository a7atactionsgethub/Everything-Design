Large-type footers are basically:

> typography-first cinematic ending sections.

The references consistently show:

* oversized type as the main visual anchor
* reduced UI clutter
* aggressive hierarchy
* massive spatial breathing room
* typography acting as layout, not decoration
* footer behaving like a final poster frame ([Footer][1])

And this is the proper prompt structure for it:

```txt id="g7v2mx"
LARGE TYPE FOOTER GENERATION MODE

Design a typography-dominant footer that uses large-scale text as the primary visual and emotional element of the closing experience.

The footer should feel bold, cinematic, architectural, and compositionally powerful.

Typography is not decoration here.
Typography IS the layout.

The footer must create a strong final visual impression while remaining deeply integrated into the current project's design system.

Before generating:
- analyze the project's visual identity
- study existing typography hierarchy
- detect spacing rhythm and layout density
- understand the current grid and alignment philosophy
- preserve the project's established atmosphere and implementation style

The footer should feel:
- typographic
- cinematic
- premium
- bold
- restrained
- editorial
- architectural
- visually commanding

Focus heavily on:
- oversized typography
- compositional balance
- large-scale text rhythm
- negative space
- typography as structure
- horizontal and vertical alignment
- visual pacing
- dramatic but controlled hierarchy
- premium restraint
- immersive spacing

Typography should:
- dominate the visual composition
- create emotional impact
- act as a structural design element
- feel intentional and art-directed
- maintain readability despite scale
- reinforce the project's identity

Possible large-type approaches:
- massive single-line statements
- oversized split typography
- layered type compositions
- cinematic call-to-action typography
- ultra-large brand/title treatment
- editorial-inspired footer layouts
- typography interacting with grid systems
- cropped or partially overflowing type treatments
- monochrome contrast-focused compositions

Layout behavior should:
- allow typography to drive spacing decisions
- feel spacious and intentional
- avoid clutter and dense utility layouts
- create a strong visual landing zone at the bottom of the page

Avoid:
- generic multi-column SaaS footers
- excessive utility links
- small weak typography
- random gradients
- glassmorphism
- startup-style CTA spam
- social icon overload
- visual clutter
- overcomplicated interactions
- decorative typography without structural purpose

Interactions should:
- remain subtle and premium
- reinforce typography dominance
- avoid distracting animation
- support cinematic presentation

The footer must feel like a professionally art-directed ending frame rather than a standard website footer.

Do not redesign the website.
Do not introduce unrelated aesthetics.
Do not break the existing visual language.

Output:
1. Large-type footer concept
2. Typography composition strategy
3. Layout structure
4. Visual hierarchy logic
5. Spacing philosophy
6. Interaction behavior
7. Responsive typography scaling
8. Architectural integration notes
9. Implementation considerations
```
